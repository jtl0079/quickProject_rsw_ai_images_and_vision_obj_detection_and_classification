# Shape Detection and Classification System

## Project Overview

This project aims to develop a Shape Detection and Classification System capable of identifying and classifying three basic geometric shapes:

- Circle
- Triangle
- Square

The system combines traditional Image Processing techniques and Deep Learning approaches to compare their performance in object detection and classification tasks.

---

## Project Objectives

### Main Objectives

1. Detect geometric shapes in images.
2. Classify detected shapes correctly.
3. Compare traditional Computer Vision methods with Deep Learning methods.
4. Develop a simple application for demonstration purposes.

### Learning Outcomes

- Understand Image Processing workflows.
- Apply Object Detection and Classification concepts.
- Gain experience with OpenCV and YOLO.
- Evaluate and compare different detection approaches.
